# QueryDesk

Internal operational ticketing and query management platform for visa and application support teams.

## Development

This prototype has been converted into a React + Vite frontend application. It remains client-side only: there is no backend, authentication service, Redux store, or TypeScript layer yet.

Run it locally with:

```bash
npm install
npm run dev
```

The demo state is stored in `localStorage` under `querydesk_demo_data`.
